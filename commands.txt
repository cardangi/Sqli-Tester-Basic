Welcome to the command/argv list.


-definir

Defines your target, to check if it's compatible to basic SQL Inject. Didn't save LOGS. Just appear in the windows-prompt/ terminal if it's compatible

-definirVarios
Defines more than 1 target, to check if they are compatible to basic SQL Inject. Didn't save LOGS. Just appear in the windows-prompt/terminal if it's compatible

-definir -LOG
Defines your target, to check if it's compatible to basic SQL Inject, but saves LOGS in a TXT file.

-ajuda
Read the commands.txt INSIDE the program, in a windows-prompt/terminal
